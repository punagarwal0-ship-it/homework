# Project Statement

## Problem Statement
In many small library, the librarian is writing everything in a notebook which is very slow process. doing calculation for late fine is hard because sometimes months have different days and leap year is confusing. also paper records can get lost or damaged easily so we need digital way.

## Scope of the project
This project is make only for small library or personal collection. It handles the basic work like issuing book and returning it. It does not have advance features like barcode scanner or internet database right now, it is just for local computer.

## Target users
* School Librarians who want simple software.
* Students who manage class library.
* Anyone who has many books at home and want to track them.

## High-level features
* **Book Management:** allow user to add new book to the list.
* **Transaction:** logic to give book and take it back from student.
* **Fine Calculation:** it calculates how many days is overdue and multiplies by fine amount.
* **Data Saving:** it saves everything in a csv file so data is safe even if we close the code.
